# react-progressive-media-loading

# Usage

```js

var ProgresssiveImg = require('react-progressive-media-loading');

var component = React.createClass({
  render: function() {
    return (
      <div>
        <ProgresssiveImg imageSrc={''} />
      </div>
    );
  }
});
```

#### 1. How to test module

```shell
$ npm run test
```

#### 2. How to build module

```shell
$ npm run build
```

#### 3. How to demo app

```shell
$ npm run demo
```
